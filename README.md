# ChinaStudyFree · 小学全科 AI 学习平台

> **一个免费、开源、纯公益的小学全科学习平台**
>
> 我们相信：**每一个中国孩子，无论身处北上广深，还是大山深处的乡村小学，都应该拥有一样好的学习资源。**
>
> 让孩子在玩中学，让 AI 方便你我他。 ❤️

---

## 🌱 项目愿景

中国的教育资源分布不均是一个长期存在的问题。一线城市的孩子可以购买数百元一套的教辅、上价格高昂的补习班；而偏远地区的孩子，往往连一本配套练习册都难以获得。

**ChinaStudyFree** 希望借助 AI 的力量，改变这件事：

- 📚 **覆盖小学全科**：语文、数学、英语、科学、道德与法治
- 🆓 **永久免费**：不收费、不卖课、不做广告、不收集隐私
- 🤖 **AI 原创题目**：基于人教版/统编版教材知识体系，由 AI 生成单元小测、综合考试与知识点全解
- 🎮 **在玩中学**：题目形式生动、即时反馈，让孩子在闯关和互动中建立对知识的兴趣
- 🌏 **服务每一个孩子**：从一年级到六年级，从城市到乡村，只要有一台能上网的设备，就能用

我们做的不是一个"产品"，而是一份可以被复制、被修改、被分发的**公共数字教育资源**。欢迎任何老师、家长、志愿者、开发者一起参与进来。

## ✨ 主要特性

- **覆盖人教版/统编版核心学科**
  - 数学（人教版）
  - 语文（统编版）
  - 英语（人教版 PEP，三年级起）
  - 科学（人教版）
  - 道德与法治（统编版）
- **三种学习模式**
  - **单元小测**：跟着课本章节走，学完一课就能练
  - **综合考试**：模拟期中/期末，检验阶段性学习成果
  - **知识点全解**：对每个知识点配有清晰讲解、例题和变式
- **AI 驱动的内容生成 Pipeline**：从教材大纲 → 知识点 → 原创题目 → 解析，全流程自动化
- **现代 Web 前端**：基于 Next.js + React，响应式设计，手机、平板、电脑都好用
- **离线可部署**：生成的内容是静态 JSON，可以打包部署到任何静态托管，甚至校园内网

## 🏗️ 项目结构

```
ChinaStudyFree/
├── pipeline.py              # 核心数据生成 Pipeline（PDF → 大纲 → 知识点 → 题库）
├── prompts.py               # AI Prompt 模板（按学科定制）
├── subjects.py              # 学科/版本/年级配置
├── scripts_cleanup_quizzes.py
├── plan.md                  # 项目技术设计与路线图
├── output/                  # Pipeline 产出（教材大纲、知识点、题库 JSON）*不入库*
└── frontend/                # Next.js 前端
    ├── src/
    │   ├── app/             # 路由与页面
    │   ├── components/      # UI 组件
    │   ├── lib/             # 数据加载与工具
    │   ├── store/           # 状态管理（Zustand）
    │   └── types/
    ├── scripts/build-data.ts
    └── public/
```

> 📌 `output/` 目录体积较大（包含原始教材和生成数据），未纳入 Git 仓库，需通过 Pipeline 本地生成或从单独的数据发布渠道获取。

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/wuwangzhang1216/ChinaTextbookStudyFree.git
cd ChinaTextbookStudyFree
```

### 2. 运行前端

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:3000 即可。

### 3. （可选）运行数据生成 Pipeline

Pipeline 会从人教版/统编版教材 PDF 中提取知识大纲，并用 AI 生成原创练习题。

```bash
# 在项目根目录
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt  # 如无，请参考 pipeline.py 的 import

# 配置环境变量（复制 .env.example）
cp .env.example .env
# 填入你的 API Key

python pipeline.py
```

## 📖 教材来源与版权

本项目引用的教材结构来自开源项目 [TapXWorld/ChinaTextbook](https://github.com/TapXWorld/ChinaTextbook)，版本选用为：

| 学科 | 版本 |
| ---- | ---- |
| 数学 | 人教版 |
| 语文 | 统编版（部编版） |
| 英语 | 人教版 PEP（三年级起） |
| 科学 | 人教版 |
| 道德与法治 | 统编版 |

**版权策略**：

1. **不搬运教材原文**，只提取知识大纲与章节结构
2. **所有题目由 AI 原创生成**，基于知识点而非教材原题
3. 如相关版权方认为本项目存在任何问题，请与我们联系，我们会第一时间响应处理
